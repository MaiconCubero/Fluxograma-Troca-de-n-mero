# Contributing

Este repositório contém o fluxograma de controle de troca de número.  
Se você identificar melhorias, inconsistências ou sugestões, faça:

1. Fork do repositório
2. Crie uma branch com a mudança (ex: `fix/update-flow`)
3. Abra um Pull Request descrevendo a alteração
4. Aguarde revisão do mantenedor

Contribuições serão analisadas caso a caso.
